
# Grizzly ALT Frontend

> A fun project to learn React.js

## Pre requisites

Have [Home Brew](https://brew.sh/) installed.

Make sure you have node installed:
```sh
brew install node
```


## Installation

Clone the project:

```sh
git clone https://github.com/CognizantGrads-TeamALT/grizzly-frontend.git
```

```sh
npm install
``` 

```sh
npm start
```

```sh
npm run local
``` 

Find the output at:
```sh
localhost:3000/
```
